Tabella

I colori per le tabelle sono impostati nel file customizzato di bootstrap: bootstrap_xxx.css.
Nel file lombard_xxx.css è stato specificato solo il colore di sfondo per il tag th che il customizzatore di bootstrap non permette di impostare.

Piccole modifiche sono state inserite per il tool di paginazione 

.pagination {
  margin-top: 0;
}
.pagination > li > a:hover {
  background-color: #c2ddc0;
}
.pagination > .active > a {
  font-weight: bold;
}

Accessibilità

Paginatore

<nav>
  <ul class="pagination pull-right" aria-label="Paginatore">
    <li><a href="#" aria-label="Pagina precedente">«</a></li>
    <li><a href="#">1</a></li>
    <li class="active"><a href="#">2<span class="sr-only">(attuale)</span></a></li>
    . . .
    <li><a href="#" aria-label="Pagina successiva">»</a></li>
  </ul>
</nav>
In grassetto sono qui evidenziati attributi e tag necessari:
  • L'attributo aria-label fa si che gli screen-reader pronuncino le parole: "Paginatore", "Pagina precedente/successiva" anche se non compaiono a video.
  • Il tag span con la classe sr-only, specifica di bootstrap, è da inserire per segnalare il numero di pagina corrente.
