Pulsanti

Per la resa grafica dei pulsanti è stata creata una classe aggiuntiva: btn-lispa.
In questo modo resta possibile anche utilizzare tutti i tipi di pulsanti previsti da bootstrap.

La classe btn-lispa, come le altre classi btn di bootstrap funzionano non solo sui tag <button>, ma anche sui tag <a>, come nel caso del terzo "pulsante" proposto nell'esempio.

Accessibilità

Nel caso si utilizzi il tag <button>, è inutile specificare role="button". 

<button type="submit" class="btn btn-lispa">Filtra</button>

Se invece si usa il tag <a> diventa necessario specificare il ruolo perchè gli screen reader identifichino come "bottone" il link. 

<a class="btn btn-lispa" role="button">Premere</a>
