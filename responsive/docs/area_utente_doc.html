Area utente

Per l'area utente si usa un div della classe row di bootstrap, cui si aggiunge la classe specifica area_utente.
E' pensata per avere sempre il nome dell'utente a sinistra e il link di logout a destra.
Facoltativamente possono essere aggiunti dei link all'interno di un tag nav con classe tools_utente.

Il nome_utente, per gli schermi particolarmente piccoli, ha una lunghezza massima di 200 pixel, dopo di che viene abbreviato coi puntini per lasciare spazio al link di logout. 

.area_utente .nome_utente {
  max-width: 200px;
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
@media (min-width: 500px) {
   .area_utente .nome_utente {
      max-width: none;
   }
}
I tools_utente non vengono visualizzati sugli smartphone (classe hidden-xs) sul tag nav che li contiene.

Accessibilità

<a class="pull-right" href="#" role="button">
   <span class="glyphicon glyphicon-off white" role="presentation"></span> Logout
</a>
<nav class="tools_utente hidden-xs pull-right" role="menu">
   <a href="#" role="menuitem"><span class="glyphicon glyphicon-cog white" role="presentation"></span> Modifica profilo</a>
   . . .
  • Il tasto di Logout viene identificato come button in quanto produce un'azione immediata.
  • L'attributo role="presentation" fa in modo che le tecnologie assistive ignorino queste "icone" di puro effetto visivo.
  • Il tag nav con classe tools_utente viene indicato come menu, in quanto le sue voci dovrebbero compiere delle azioni (anzichè essere dei link di pura navigazione).
    Di conseguenza i link in esso contenuti hanno ruolo menuitem.
