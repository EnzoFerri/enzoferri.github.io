Barra istituzionale

Questo template può essere utilizzato con un numero di modifiche minimo.
Devono essere impostati:

  1. testo del titolo
  2. sottotitolo
  3. numero verde
  4. voci di navigazione.

Il sottotitolo è facoltativo. Per le regole di composizione del titolo si veda il paragrafo delle linee guida: "Costruzione del nome dell'applicativo".

Secondo le linee guida, per voci di navigazione di servizio si intendono per esempio la logout, il ritorno alla home page d’origine, il canale di assistenza.

A tutte queste diciture è applicato dai CSS un text-transform: uppercase che rende i caratteri maiuscoli.

Il layout è fluido, quindi si adatta alla larghezza del video.
La testata presentata nell'esempio può essere ridotta fino alla larghezza di circa 700 pixel senza che le scritte vadano a capo, anche se la cosa dipende da:

  • i font installati nel sistema operativo del client
  • le dimensioni dei font scelte dall'utente all'interno del browser
  • la lunghezza del titolo.

Le linee guida (vedi al paragrafo: "risoluzione") consigliano di mantenere una larghezza effettiva non superiore ai 980 pixel per evitare la comparsa della scroll bar orizzontale sulla risoluzione minima considerata: 1024x792.

Il link <a href="#titolo_pagina" class="non_visibile">salta al contenuto</a> che precede tutti gli altri tag, è un accorgimento utile agli utenti non vedenti.
Dà la possibilità, navigando la pagina con uno screeen reader, di evitare la rilettura di tutte le parti dell'intestazione e dell'eventuale menù verticale, passando al contenuto più specifico della pagina.
Presuppone l'esistenza dell'id="titolo pagina" come previsto dal componente titolo_pagina illustrato più avanti.
La class="non_visibile" fa si che non venga visualizzato, così da non modificare l'aspetto grafico della testata.
