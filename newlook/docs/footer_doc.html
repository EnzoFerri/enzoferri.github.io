Footer

Queste regole nel lomb_xxx.css fanno sì che il footer sia sempre posizionato in fondo allo schermo, anche quando la pagina ha poco contenuto, ed abbia un'altezza minima di 30 pixel. 

html {
  position: relative;
  min-height: 100%;
}
body {
  /* Margin bottom by footer height */
  margin-bottom: 30px;
}

La dicitura riguardante il copyright viene accorciata per gli smartphone.
Nei casi in cui non sia sufficiente, per la presenza di molti link sul footer, si può aggiungere la classe bootstrap hidden-xs anche allo <span>&copy; Copyright Regione Lombardia</span>.

Accessibilità

Il tag nav contenente le voci di navigazione ha il proprio attributo role="navigation". 

<nav class="voci_navigazione" role="navigation">
   <a href="#">Voce 01</a>
   . . .
